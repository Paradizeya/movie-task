# Установка

## Клонирование репозитория

```
git clone https://github.com/Paradizeya/movie-task.git
```

## Установить пакеты

```
npm install
```

## Переменные окружения
Создайт файл ```.env``` и добавьте туда: 

```
VITE_API_KEY=ВАШ_КЛЮЧ
```
# Инфо
